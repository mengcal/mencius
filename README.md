# Mencius
没基础的瞎玩
这个是我一个非专业没基础的好奇了半个月的开源软件安装，被坑的多了，想要的一个设定
用大模型辅助我这个不会的人，经常被坑，一句没有VPN或者代理用不了，不告诉，非要带着我折腾，几个小时过去了，换个窗口才知道
所以这个是在OPEN WEBUI里，添加一个函数，实现多个大模型互动，挑刺，最后总结的一个流程，兼听则明偏听则暗，
感谢智谱agentmore里的全栈开发帮我把想法落实
感谢百炼提供的免费额度，让我验证了一下，的确挺好玩的
就不贴我试验的对话内容了，感兴趣的可以自己试试
反正我看不懂函数内容，我只看到形式还可以
不预设任何大模型的角色，背景，就是裸奔上阵，这个我要求的是不设上限，你有多少模型你自己上多少
我不会，不懂，我只看模型说的是不是真的有道理，能真的实现，
